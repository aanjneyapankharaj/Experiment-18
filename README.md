Overview

This experiment demonstrates the use of advanced and interactive data visualisation techniques to represent real-world scenarios. Various types of plots are implemented using Python libraries to convey complex relationships, hierarchies, and patterns in a clear and meaningful way.

Tools and Technologies Used
Python
Pandas
NumPy
Matplotlib
Plotly
SciPy
Matplotlib-Venn
Objectives
To understand different types of real-world visualisations
To represent hierarchical, relational, and multidimensional data
To create interactive and visually appealing charts
To analyse data patterns effectively using graphical methods
Visualisations Implemented
1. Treemap (Company Department Budget)

A treemap is used to represent hierarchical data as nested rectangles.

Displays budget allocation across different departments
Helps in comparing proportions visually
2. Dendrogram

A dendrogram represents hierarchical clustering.

Shows relationships between data points
Useful in grouping similar data using clustering techniques
3. Venn Diagram

A Venn diagram illustrates set relationships.

Displays common and unique elements between two sets
Useful for logical comparisons and overlap analysis
4. Sankey Diagram (Student Flow)

A Sankey diagram visualises flow between stages.

Represents student progression from admission to placement
Highlights flow quantity using link thickness
5. 3D Scatter Plot (Student Performance)

A 3D scatter plot represents three variables simultaneously.

Study Hours, Marks, and Attendance are analysed together
Helps identify trends and correlations in student performance
6. Radar Chart (Skill Assessment)

A radar chart displays multivariate data in a circular form.

Represents student skill levels across multiple domains
Useful for performance comparison and assessment
Key Insights
Visualisations provide intuitive understanding of complex datasets
Interactive charts enhance user engagement and data exploration
Different chart types serve different analytical purposes
Multidimensional data can be effectively analysed using advanced plots
Conclusion

This experiment highlights the importance of visualisation in data analysis. By using various plotting techniques, complex data can be simplified and interpreted efficiently. Interactive visualisations further improve the ability to explore and communicate insights clearly.
